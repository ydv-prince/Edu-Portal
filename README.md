# 🎓 EduPortal – Online Course Management System (Backend)

EduPortal is a scalable backend system for an online learning platform built using **Spring Boot**. It supports user authentication, course management, enrollment, and role-based access control for Students, Teachers, and Admins.

---

## 🚀 Tech Stack

- **Language:** Java  
- **Framework:** Spring Boot, Spring MVC  
- **Security:** Spring Security, JWT Authentication, BCrypt  
- **Database:** MySQL  
- **Build Tool:** Maven  
- **API Testing:** Postman  

---

## 📌 Features

### 🔐 Authentication & Security
- User registration and login
- JWT-based authentication
- Role-Based Access Control (RBAC)
- Password encryption using BCrypt

### 👤 User Management
- Profile management
- Role support: Student / Teacher / Admin

### 📚 Course Management
- Create, update, delete courses
- Course publishing system
- Categorization and filtering

### 🎓 Enrollment System
- Course enrollment
- Track enrolled courses
- Prevent duplicate enrollments

### 📊 Admin Dashboard
- Manage users
- Monitor courses
- View platform analytics
